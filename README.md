# E-commerce Testing Project

## Project Overview
This project demonstrates end-to-end manual testing of an e-commerce application including login, product, cart, and checkout modules.

## Modules Covered
- Login
- Product
- Cart
- Checkout

## Test Artifacts
- Test Cases
- Bug Report
- Test Summary
- Test Data

## Tools Used
- Excel
- BrowserStack

## Test Approach
- Designed positive and negative test cases
- Executed test cases and recorded results
- Identified defects and reported bugs
- Categorized bugs based on severity
